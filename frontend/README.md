# Investor Radar

A modern stock screener with **Radar Stock Pulse Rating** - an intelligent system for identifying promising investment opportunities.

## Features

- **Advanced Stock Screening**: Filter stocks by multiple criteria including price, market cap, PE ratio, and technical indicators
- **Radar Stock Pulse Rating**: Proprietary rating system that combines fundamental and technical analysis to evaluate stock health and momentum
- **Real-time Data**: Access current market data and historical trends
- **Interactive Dashboard**: Visualize market trends and individual stock performance
- **Watchlist Management**: Save and track your favorite stocks

## Getting Started

### Prerequisites
- Node.js 14+
- npm or yarn

### Installation
cd backend
py app.py

cd frontend
npm run dev


### Usage
1. Navigate to the screener
2. Apply filters based on your investment strategy
3. Review Radar Stock Pulse Ratings for each stock
4. Add promising stocks to your watchlist

## Technology Stack

- React.js
- Tailwind CSS / Material-UI
- Real-time market data API integration
